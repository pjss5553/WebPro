R이란?

R이란? "R은 통계 분석에 특화된 프로그래밍 언어로, 데이터 분석 시 많이 사용합니다. 프로그래밍 언어는 사용자가 직접 모든 걸 명령어로 입력하고 실행해야 하므로 상당히 불편합니다. R은 RGui라는 프로그램으로 작동합니다. 그리고 R 스튜디오는 R을 보다 효과적이고 편리하게 사용할 수 있도록 돕는 별도의 GUI 프로그램으로 RGui보다 편하게 데이터 분석을 할 수 있습니다."

변수란? "프로그램에서 어떤 값을 저장하는 저장소나 보관 상자의 개념이며, 변수에 저장된 값은 언제라도 변경 가능 변수의 자료형은 어떤 값을 저장하는가에 따라 유동적으로 바뀔수도 있습니다"

리스트란?"서로 다른 자료형의 값들을 1차원 배열에 저장하고 다룰 수 있도록 해주는 수단이고, 벡터와 유사하지만 차이점은 이름을 지정할 수 있다"

매트릭스란?데이터 테이블의 모든 셀의 값들이 동일한 자료형이고 1차원 데이터는 ‘몸무게’ 데이터와 같은 단일 주제의 데이터 → 벡터 2차원 데이터는 ‘키’, ‘몸무게’, ‘나이’ 와 같은 여러 주제의 데이터이다

데이터프레임이란?숫자형 벡터, 문자형 벡터 등 서로 다른 형태의 데이터를 2차원 데이터 테이블 형태로 묶을 수 있는 자료구조이고 외관상으로는 매트릭스와 차이가 없지만,매트릭스에 저장되는 모든 값들이 동일한 자료형인 것과는 달리 데이터 프레임에는 서로 다른 자료형의 값들이 함께 저장된다.

자료의 종류자료에는 범주형 자료와 연속형 자료가 있고 범주형 자료는 범주형 자료의 값들은 기본적으로 숫자로 표현할 수 없고, 대소(大小) 비교나 산술 연산이 적용되지 않는다, 연속형 자료는 연속형 자료의 값들은 대소 비교가 가능하고, 평균, 최댓값, 최솟값과 같은 산술 연산이 가능하다.
